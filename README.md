# Calculator Web App

A simple and responsive **Calculator Web Application** built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations such as addition, subtraction, multiplication, division, and percentage calculations.

## Features

* Clean and modern calculator UI
* Basic arithmetic operations (+, −, ×, ÷)
* Percentage calculation
* Clear (C) button to reset the display
* Backspace button to remove the last digit
* Decimal number support
* Responsive layout

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Google Fonts (Roboto)

## Project Structure

```
calculator/
│
├── index.html   # Main HTML file
├── style.css    # Styling for the calculator
└── script.js    # Calculator logic
```

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Start using the calculator.

## Purpose

This project was created to practice **DOM manipulation, event handling, and basic JavaScript logic** while building a functional UI component.
